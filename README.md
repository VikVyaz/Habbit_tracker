# Трекер привычек

1. Сделать свои `.env` и `Dockerfile` из `samles`
2. Запустить `Docker`
2. Прописать в консоль проекта `docker-compose up -d --build`
3. Проверить контейнеры можно через `docker-compose ps`, Логи - `docker-compose logs <название сервиса>`
4. Можно заходить на `localhost:8000` и пользоваться
5. Если надо очистить БД, удалить контейнеры и тд - `docker-compose down -v`





### Отслеживания привычек
* интегрирование информирование через Telegram о создании, изменении и удалении привычки
* интегрирование информирование через Telegram о привычке за час до ее начала
* документирование через swagger/redoc
